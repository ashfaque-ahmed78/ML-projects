# 🚗 Car Price Prediction App

## 📌 Overview
The **Car Price Prediction App** is a Machine Learning–based project that predicts the **selling price of a car** based on its specifications and features.  
The model is trained using **Random Forest Regression** and deployed using **Streamlit** for an interactive web experience.

---

## ✨ Features
✅ Predicts car price instantly  
✅ Trained ML model with feature scaling  
✅ Interactive Streamlit web app  
✅ Complete EDA and documentation  
✅ Real-world dataset  

---

## 📊 Input Features
The prediction is based on the following car details:

- 🚘 **Car Brand / Model**
- 📅 **Year of Purchase**
- 🛣️ **Present Price**
- 📉 **Kilometers Driven**
- ⛽ **Fuel Type**
- ⚙️ **Transmission Type**
- 👤 **Number of Owners**

---
