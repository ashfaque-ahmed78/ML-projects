# 📄 Resume Screening System (Machine Learning)

## 🔍 Overview
The Resume Screening System is a Machine Learning-based application designed to automate the process of classifying resumes into predefined job categories. This system assists recruiters in efficiently filtering and shortlisting candidates by analyzing resume content.

---

## 🎯 Objective
- Automate the manual resume screening process  
- Improve recruitment efficiency  
- Enable faster candidate shortlisting using ML  

---

## ⚙️ System Workflow
1. Upload a resume (PDF, DOCX, or TXT format)  
2. Extract textual content from the file  
3. Perform text preprocessing (cleaning, tokenization, stopword removal)  
4. Transform text using TF-IDF Vectorization  
5. Predict the job category using a trained Machine Learning model  

---

## 🚀 Features
- Supports multiple file formats (PDF, DOCX, TXT)  
- Real-time prediction  
- User-friendly interface (Streamlit)  
- Fast and efficient classification  

---

## 🛠️ Tech Stack
- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, NLTK  
- **Framework:** Streamlit  

---

## 🤖 Machine Learning Details
- **Model:** Naive Bayes / Logistic Regression  
- **Feature Extraction:** TF-IDF Vectorizer  
- **Saved Model:** `clf.pkl`
